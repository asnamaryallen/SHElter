# SHElter - Women's Safety SOS App

A decoy application designed to provide emergency assistance while appearing as a harmless Quote app.

## Features
* **Decoy UI:** Looks like a daily inspiration app.
* **Triple Tap Trigger:** Activates SOS protocol via 3 rapid taps.
* **Stealth Call:** Automatically dials a trusted contact for audio monitoring.
* **Continuous Tracking:** Sends real-time Google Maps links via SMS as the user moves.
* **Fake Power-Off:** Simulates a black screen to prevent intruder interference.

## Setup
1. Clone the repository.
2. Build the APK using Android Studio.
3. **Long-press** the "Daily Inspiration" title to set your trusted contact number.
4. Ensure **Location, SMS, and Phone** permissions are manually granted in System Settings.
